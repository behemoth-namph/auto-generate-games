# sudoku

## Description

Sudoku is a logic puzzle where you fill a 9x9 grid with digits 1-9 so that each row, column, and 3x3 box contains every number exactly once. Tackle it at your own pace or challenge yourself with timed modes, aided by hints and smooth, responsive controls. With PixiJS-powered 2D rendering, expect crisp visuals, fluid interactions, and a clean, distraction-free grid.
